# BeamBeats
Source code for the BeamBeats Arduino project

# Schematics

LDR 5mm GL5506
- Light resistance (10 Lux) (KΩ): 2-5
- Dark resistance (MΩ): 0.2

Wiring

    +5V ---2.2k---+---LDR---GND

MIDI DIN plug

    \ / -------220--- +5V
    o - GND
    / \ Tx

Hall Sensor

U18/US1881/OH188/1881

     /---------\
     +---------+
     |    |    |
    IN   GND  +5V
     |         |   
     +--10k----+
